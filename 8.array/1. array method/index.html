Array methods

push : add to end

unshift : add to start

pop : delete from end and return it

shift : delete from start and return it

indexOf : return index of something

includes : search for a value and return true and false after searching

concat: used for merge two arrays

reverse: reverse an array

slice : copies a portion of array

splice: removes/replaces/add element

sort : sorts an array (it doesnot work well for the numbers)
